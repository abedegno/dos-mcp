# dos-mcp

An MCP server that lets AI agents drive, observe, and inspect DOS programs via js-dos.

Status: Phase 1 in development. See [`docs/superpowers/specs/`](docs/superpowers/specs/).

## Planned Phase 1 tools

- `load_bundle`, `shutdown`, `wait`
- `send_keys`, `send_key_sequence`, `send_click`, `move_mouse`
- `screenshot`, `get_status`
- `fs_read`, `fs_write`, `fs_list`, `fs_delete`, `fs_push_dir`, `fs_pull_dir`, `fs_sync`

## Install (once shipped)

```
npm install -g dos-mcp
```
